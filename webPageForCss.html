CASCADING STYLE SHEETS

WHAT IS CSS!..?

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML. It is a fundamental technology of the World Wide Web, working alongside HTML and JavaScript to create interactive and visually appealing web pages

Color

Defines the text color of an element.

Font Size

Controls the size of the text.

Background Color

Sets the background color of an element.
